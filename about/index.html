关于本站

这是我在学习过程中搭建起来的个人博客网站。

  • 基于 github 的 page 服务，使用hexo博客框架，主题是next，本地环境是Nodejs，平时推送使用Git。
  • 域名备案通过后，部署到阿里云服务器上。
  • 添加了七牛云的图床，采用阿里云的CDN加速以提高网站的加载速度。
  • SEO优化正在学习中。。。。
  • 网站美化并不充分，追求加载速度。
  • 本博客存储一些学习笔记，就当做是个人空间。和大家一起分享。
